## Clone the repository
git clone https://github.com/saad2786/roxiler-task.git\
`cd roxiler-task`

## Install server dependencies
`cd ./api`\
`npm install`

## Install client dependencies
`cd ../client`\
`npm install`

## Run the server (from api directory)
`npm start`

## Run the client (from client directory)
`npm run dev`
